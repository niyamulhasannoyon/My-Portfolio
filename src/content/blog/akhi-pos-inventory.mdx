## The challenge

Retail operations were drowning in spreadsheets and legacy POS tools that couldn't keep pace with real-time inventory needs. Sales entries lagged, stock counts were never accurate, and the team needed a system that worked reliably across multiple locations.

## What I built

A cloud-native POS dashboard built with React and Node.js, backed by a MongoDB database architecture designed for real-time accuracy. Cloudflare routing ensures low-latency access from any location, and the intuitive interface makes sales entry and stock tracking effortless for staff.

## Outcomes

- Inventory sync improved by **85%** — stock levels now update in real time across all locations
- Sales entry time reduced by **62%** with the streamlined POS interface
- System uptime maintained at **99.9%** thanks to robust cloud architecture
- Manual data entry errors virtually eliminated through automated validation

> "We went from spreadsheet chaos to real-time inventory accuracy in one deployment." — Operations Lead, Akhi POS
